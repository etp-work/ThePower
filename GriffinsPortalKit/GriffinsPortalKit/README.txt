To be able to run the Native Container:

1. Download http://ftp.mozilla.org/pub/mozilla.org/xulrunner/releases/16.0/runtimes/xulrunner-16.0.en-US.win32.zip
2. Unzip all files to xulrunner directory under the application directory (which has GriffinsPortalKit.exe)
3. Run GriffinsPortalKit.exe
